5. 最长回文子串

动态规划

  • 如果一个字符串的头尾两个字符都不相等，那么这个字符串一定不是回文串；

  • 如果一个字符串的头尾两个字符相等，才有必要继续判断下去。

    • 如果里面的子串是回文，整体就是回文串；

    • 如果里面的子串不是回文串，整体就不是回文串。

状态：dp[i][j] - s[i:j] 是否为回文串
状态转移方程：dp[i][j] = s[i] == s[j] and dp[i + 1][j - 1]

public String longestPalindrome(String s) {
    int len = s.length();
    if (len == 0) {
        return "";
    }
    boolean[][] dp = new boolean[len][len];
    // 起点
    int start = 0;
    // 回文长度
    int maxLen = 1;
    char[] chars = s.toCharArray();
    for (int j = 1; j < len; j++) {
        for (int i = 0; i < j; i++) {
            // 两侧不等，一定无法构成回文
            if (chars[i] != chars[j]) {
                dp[i][j] = false;
            } else {
                    // i, j 之间的子串只有 0 个或 1 个字符
                if (j - i <= 2) {
                    dp[i][j] = true;
                } else {
                    // 两侧相等，看子串是不是回文
                    dp[i][j] = dp[i + 1][j - 1];
                }
            }
            // 更新最长回文位置
            if (dp[i][j] && j - i + 1 > maxLen) {
                maxLen = j - i + 1;
                start = i;
            }
        }
    }
    return s.substring(start, start + maxLen);
}

中心扩散

尝试以每个字符和每个字符间的间隙为中心，向两端扩散。

public String longestPalindrome(String s) {
    char[] str = s.toCharArray();
    int len = str.length;
    int maxLen = 1;
    int begin = 0;
    for (int i = 0; i < len - 1; i++) {
        // 奇数扩散，以 s[i] 为中心
        int[] res1 = centerSpread(str, i, i);
        // 偶数扩散，以 s[i], s[i + 1] 为中心
        int[] res2 = centerSpread(str, i, i + 1);
        int[] res = res1[1] > res2[1] ? res1 : res2;
        // 更新长度
        if (res[1] > maxLen) {
            maxLen = res[1];
            begin = res[0];
        }
    }
    return s.substring(begin, begin + maxLen);
}

private int[] centerSpread(char[] str, int left, int right) {
    int len = str.length;
    // 结束循环时，str[left] != str[right] 或越界
    // 取 str(left:right)
    while (left > 0 && right < len) {
        if (str[left] != str[right]) {
            break;
        }
        left--;
        right++;
    }
    return new int[]{left + 1, right - left - 1};
}
