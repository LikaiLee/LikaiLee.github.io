插入排序

1. 直接插入排序

将每个元素依次插入前面已经排序好的序列。需要反复把已排序元素向后移，为新元素提供插入空间。

  • 空间复杂度：O(1)O(1)O(1)
  • 时间复杂度：每趟操作都要比较关键字和移动元素，而比较次数和移动次数取决于待排序表的初始状态。
    • 最好情况：元素已经有序，每个元素只需比较一次 - O(n)O(n)O(n)
    • 最坏情况：元素逆序 - O(n2)O(n^2)O(n2)
    • 平均：O(n2)O(n^2)O(n2)
  • 稳定性：稳定 - 每次插入总是先比较再移动，不会出现相同元素的相对位置变化。
  • 适用性：适用于顺序存储和链式存储的线性表
private void insertSort(int[] arr) {
    int cur, j;
    // 位置 0 作为第一个有序元素
    for (int i = 1; i < arr.length; i++) {
        // 当 cur < 前一个元素，进行排序
        if (arr[i] < arr[i - 1]) {
            cur = arr[i];
            // 后移直到 cur >= arr[j]
            for (j = i - 1; j >= 0 && arr[j] > cur; --j) {
                arr[j + 1] = arr[j];
            }
            // 插入当前元素
            arr[j + 1] = cur;
        }
    }
}

2. 折半插入排序

在查找有序表时使用折半查找实现。

仅仅减少了比较次数，其他与直接插入相同。

private void insertSort(int[] arr) {
    int low, high;
    int cur;
    for (int i = 1; i < arr.length; i++) {
        cur = arr[i];
        low = 0;
        high = i - 1;
        // low = high + 1 时结束，即 low 为最终插入位置
        while (low <= high) {
            int mid = (low + high) >> 1;
            if (arr[mid] > cur) {
                high = mid - 1;
            } else {
                low = mid + 1;
            }
        }
        // 移动 low 直到 i 位置
        for (int j = i - 1; j >= low; j--) {
            arr[j + 1] = arr[j];
        }
        arr[low] = cur;
    }
}

3. 希尔排序

根据增量序列，对子序列进行排序。

  • 空间复杂度：O(1)O(1)O(1)
  • 时间复杂度：依赖于增量函数
    • 最好：O(n)O(n)O(n)
    • 最坏：O(n2)O(n^2)O(n2)
    • 平均：O(n1.3)O(n^{1.3})O(n1.3)
  • 稳定性：不稳定 - 当相同元素被划分到不同的子序列时，相对位置可能改变。
  • 适用性：顺序存储的线性表
private void shellSort(int[] arr) {
    int n = arr.length;
    int cur;
    int j;
    for (int dk = n / 2; dk >= 1; dk /= 2) {
        // 对增量 dk 内的每个元素排序
        for (int i = dk + 1; i < n; i++) {
            if (arr[i] < arr[i - dk]) {
                cur = arr[i];
                // 寻找插入位置
                for (j = i - dk; j >= 0 && arr[j] > cur; j -= dk) {
                    arr[j + dk] = arr[j];
                }
                arr[j + dk] = cur;
            }
        }
    }
}

交换排序

1. 冒泡排序

两两比较相邻元素的值，若逆序则交换，一趟冒泡结束后，当前步骤的最大元素到达最终位置。

  • 空间复杂度：O(1)O(1)O(1)
  • 时间复杂度：
    • 最好：初始序列有序 - O(n)O(n)O(n)
    • 最坏：O(n2)O(n^2)O(n2)
    • 平均：O(n2)O(n^2)O(n2)
  • 稳定性：稳定 - 元素相等时不进行交换
private void bubbleSort(int[] arr) {
    for (int i = 0; i < arr.length - 1; i++) {
        for (int j = 0; j < arr.length - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                int t = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = t;
            }
        }
    }
}

2. 快速排序

选取一个基准元素，将排序表分为两部分，该元素被放到最终位置，递归执行该过程。

  • 空间复杂度：与栈深度一致
    • 最好：O(log2(n+1))O(log_2(n+1))O(log2​(n+1))
    • 最坏：O(logn)O(logn)O(logn)
    • 平均：O(log2n)O(log_2n)O(log2​n)
  • 时间复杂度：与划分是否对称有关
    • 最好：O(nlog2n)O(nlog_2n)O(nlog2​n)
    • 最坏：有序或逆序 - O(n2)O(n^2)O(n2)
    • 平均：O(nlog2n)O(nlog_2n)O(nlog2​n)
  • 稳定性：不稳定 - 会进行左右区间的交换
private void quickSort(int[] arr, int low, int high) {
    if (low < high) {
        // pivot 的最终位置
        int pivotPos = partition(arr, low, high);
        // 对左半部分递归排序
        quickSort(arr, low, pivotPos - 1);
        // 对右半部分递归排序
        quickSort(arr, pivotPos + 1, high);
    }
}

private int partition(int[] arr, int low, int high) {
    int pivot = arr[low];
    while (low < high) {
        // 找到比 low 小的元素
        while (low < high && arr[high] >= pivot) {
            // 向前查找
            high--;
        }
        // 小元素移到 low
        arr[low] = arr[high];
        // 找到比 pivot 大的元素
        while (low < high && arr[low] <= pivot) {
            // 向后查找
            low++;
        }
        // 大元素移到 high
        arr[high] = arr[low];
    }
    // pivot 的最终位置
    arr[low] = pivot;
    return low;
}

选择排序

1. 简单选择排序

在未排序序列中找到最大（最小）值，存放到排序序列的末尾。一趟排序可以确定一个元素的最终位置。

  • 空间复杂度：O(1)O(1)O(1)
  • 时间复杂度：O(n2)O(n^2)O(n2)
  • 稳定性：不稳定
private void selectSort(int[] arr) {
    int n = arr.length;
    // 从 0 开始存放元素
    for (int i = 0; i < n - 1; i++) {
        int min = i;
        for (int j = i + 1; j < n; j++) {
            // 找到最小的元素位置
            if (arr[j] < arr[min]) {
                min = j;
            }
        }
        // 交换第 i 个元素和 min
        if (min != i) {
            swap(arr, i, min);
        }
    }
}

private void swap(int[] arr, int i, int min) {
    int t = arr[i];
    arr[i] = arr[min];
    arr[min] = t;
}

归并排序

1. 二路归并排序

将两个或两个以上的有序表组合成一个新的有序表。

  • 空间复杂度：O(n)O(n)O(n)
  • 时间复杂度：O(log2n)O(log_2n)O(log2​n) 一趟归并 O(n)O(n)O(n)，共 log2nlog_2nlog2​n 趟
  • 稳定性：稳定
private void mergeSort(int[] arr, int low, int high) {
    if (low < high) {
        int mid = (low + high) >>> 1;
        mergeSort(arr, low, mid);
        mergeSort(arr, mid + 1, high);
        merge(arr, low, mid, high);
    }
}

private void merge(int[] arr, int low, int mid, int high) {
    // 辅助数组
    int[] tmp = new int[arr.length + 1];
    for (int i = 0; i < arr.length; i++) {
        tmp[i] = arr[i];
    }
    // 合并 [low...mid] 和 [mid + 1...high]
    int i, j, k;
    for (i = low, j = mid + 1, k = i; i < mid && j < high; k++) {
        if (tmp[i] <= tmp[j]) {
            arr[k] = tmp[i++];
        } else {
            arr[k] = tmp[j++];
        }
    }
    // 合并剩下那个表剩余的部分
    while (i <= mid) {
        arr[k++] = tmp[i++];
    }
    while (j <= high) {
        arr[k++] = tmp[j++];
    }
}

总结

每一次排序之后都能确定至少一个元素位置的排序方法包括：

  1. 选择排序：每次将最大的数放到最后。所以最大的数排一次序后位置就确定了。

  2. 冒泡排序：同选择排序。每一次排序最大的值位置确定。

  3. 快排：每一次排序 pivot 的位置确定。

  4. 堆排序：每一次排序时，都是将堆顶的元素和最后一个节点互换，然后调整堆，再将堆大小减1。所以每一次排序堆顶元素确定。
