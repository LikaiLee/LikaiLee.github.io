插入排序

1. 直接插入排序

将每个元素依次插入前面已经排序好的序列。需要反复把已排序元素向后移，为新元素提供插入空间。

  • 空间复杂度：O(1)O(1)O(1)
  • 时间复杂度：每趟操作都要比较关键字和移动元素，而比较次数和移动次数取决于待排序表的初始状态。
    • 最好情况：元素已经有序，每个元素只需比较一次 - O(n)O(n)O(n)
    • 最坏情况：元素逆序 - O(n2)O(n^2)O(n2)
    • 平均：O(n2)O(n^2)O(n2)
  • 稳定性：稳定 - 每次插入总是先比较再移动，不会出现相同元素的相对位置变化。
  • 适用性：适用于顺序存储和链式存储的线性表
private void insertSort(int[] arr) {
    int cur, j;
    // 位置 0 作为第一个有序元素
    for (int i = 1; i < arr.length; i++) {
        // 当 cur < 前一个元素，进行排序
        if (arr[i] < arr[i - 1]) {
            cur = arr[i];
            // 后移直到 cur >= arr[j]
            for (j = i - 1; j >= 0 && arr[j] > cur; --j) {
                arr[j + 1] = arr[j];
            }
            // 插入当前元素
            arr[j + 1] = cur;
        }
    }
}

2. 折半插入排序

在查找有序表时使用折半查找实现。

仅仅减少了比较次数，其他与直接插入相同。

private void insertSort(int[] arr) {
    int low, high;
    int cur;
    for (int i = 1; i < arr.length; i++) {
        cur = arr[i];
        low = 0;
        high = i - 1;
        // low = high + 1 时结束，即 low 为最终插入位置
        while (low <= high) {
            int mid = (low + high) >> 1;
            if (arr[mid] > cur) {
                high = mid - 1;
            } else {
                low = mid + 1;
            }
        }
        // 移动 low 直到 i 位置
        for (int j = i - 1; j >= low; j--) {
            arr[j + 1] = arr[j];
        }
        arr[low] = cur;
    }
}

3. 希尔排序

根据增量序列，对子序列进行排序。

  • 空间复杂度：O(1)O(1)O(1)
  • 时间复杂度：依赖于增量函数
    • 最好：O(n)O(n)O(n)
    • 最坏：O(n2)O(n^2)O(n2)
    • 平均：O(n1.3)O(n^{1.3})O(n1.3)
  • 稳定性：不稳定 - 当相同元素被划分到不同的子序列时，相对位置可能改变。
  • 适用性：顺序存储的线性表
private void shellSort(int[] arr) {
    int n = arr.length;
    int cur;
    int j;
    for (int dk = n / 2; dk >= 1; dk /= 2) {
        // 对增量 dk 内的每个元素排序
        for (int i = dk + 1; i < n; i++) {
            if (arr[i] < arr[i - dk]) {
                cur = arr[i];
                // 寻找插入位置
                for (j = i - dk; j >= 0 && arr[j] > cur; j -= dk) {
                    arr[j + dk] = arr[j];
                }
                arr[j + dk] = cur;
            }
        }
    }
}

交换排序

1. 冒泡排序

两两比较相邻元素的值，若逆序则交换，一趟冒泡结束后，当前步骤的最大元素到达最终位置。

  • 空间复杂度：O(1)O(1)O(1)
  • 时间复杂度：
    • 最好：初始序列有序 - O(n)O(n)O(n)
    • 最坏：O(n2)O(n^2)O(n2)
    • 平均：O(n2)O(n^2)O(n2)
  • 稳定性：稳定 - 元素相等时不进行交换
private void bubbleSort(int[] arr) {
    for (int i = 0; i < arr.length - 1; i++) {
        for (int j = 0; j < arr.length - 1 - i; j++) {
            if (arr[j] > arr[j + 1]) {
                int t = arr[j];
                arr[j] = arr[j + 1];
                arr[j + 1] = t;
            }
        }
    }
}

2. 快速排序

选取一个基准元素，将排序表分为两部分，该元素被放到最终位置，递归执行该过程。

  • 空间复杂度：与栈深度一致
    • 最好：O(log2(n+1))O(log_2(n+1))O(log2​(n+1))
    • 最坏：O(logn)O(logn)O(logn)
    • 平均：O(log2n)O(log_2n)O(log2​n)
  • 时间复杂度：与划分是否对称有关
    • 最好：O(nlog2n)O(nlog_2n)O(nlog2​n)
    • 最坏：有序或逆序 - O(n2)O(n^2)O(n2)
    • 平均：O(nlog2n)O(nlog_2n)O(nlog2​n)
  • 稳定性：不稳定 - 会进行左右区间的交换
private void quickSort(int[] arr, int low, int high) {
    if (low < high) {
        // pivot 的最终位置
        int pivotPos = partition(arr, low, high);
        // 对左半部分递归排序
        quickSort(arr, low, pivotPos - 1);
        // 对右半部分递归排序
        quickSort(arr, pivotPos + 1, high);
    }
}

private int partition(int[] arr, int low, int high) {
    int pivot = arr[low];
    while (low < high) {
        // 找到比 low 小的元素
        while (low < high && arr[high] >= pivot) {
            // 向前查找
            high--;
        }
        // 小元素移到 low
        arr[low] = arr[high];
        // 找到比 pivot 大的元素
        while (low < high && arr[low] <= pivot) {
            // 向后查找
            low++;
        }
        // 大元素移到 high
        arr[high] = arr[low];
    }
    // pivot 的最终位置
    arr[low] = pivot;
    return low;
}

选择排序

1. 简单选择排序

在未排序序列中找到最大（最小）值，存放到排序序列的末尾。一趟排序可以确定一个元素的最终位置。

  • 空间复杂度：O(1)O(1)O(1)
  • 时间复杂度：O(n2)O(n^2)O(n2)
  • 稳定性：不稳定
private void selectSort(int[] arr) {
    int n = arr.length;
    // 从 0 开始存放元素
    for (int i = 0; i < n - 1; i++) {
        int min = i;
        for (int j = i + 1; j < n; j++) {
            // 找到最小的元素位置
            if (arr[j] < arr[min]) {
                min = j;
            }
        }
        // 交换第 i 个元素和 min
        if (min != i) {
            swap(arr, i, min);
        }
    }
}

private void swap(int[] arr, int i, int min) {
    int t = arr[i];
    arr[i] = arr[min];
    arr[min] = t;
}

2. 堆排序

使用大顶堆或小顶堆。

public int[] sortArray(int[] nums) {
    int len = nums.length;
    // 将数组整理成堆
    heapify(nums);

    // 循环不变量：区间 [0, i] 堆有序
    for (int i = len - 1; i >= 1; ) {
        // 把堆顶元素（当前最大）交换到数组末尾
        swap(nums, 0, i);
        // 逐步减少堆有序的部分
        i--;
        // 下标 0 位置下沉操作，使得区间 [0, i] 堆有序
        siftDown(nums, 0, i);
    }
    return nums;
}

/**
 * 将数组整理成堆（堆有序）
 *
 * @param nums
 */
private void heapify(int[] nums) {
    int len = nums.length;
    // 只需要从 i = (len - 1) / 2 这个位置开始逐层下移
    for (int i = (len - 1) / 2; i >= 0; i--) {
        siftDown(nums, i, len - 1);
    }
}

/**
 * @param nums
 * @param k    当前下沉元素的下标
 * @param end  [0, end] 是 nums 的有效部分
 */
private void siftDown(int[] nums, int k, int end) {
    while (2 * k < end) {
        int j = 2 * k + 1;
        // 左结点 < 右结点，j 指向右结点
        if (j < end && nums[j + 1] > nums[j]) {
            j++;
        }
        // 子结点大于父结点
        if (nums[j] > nums[k]) {
            swap(nums, j, k);
        } else {
            break;
        }
        k = j;
    }
}

private void swap(int[] nums, int i, int j) {
    int temp = nums[i];
    nums[i] = nums[j];
    nums[j] = temp;
}
  • 空间复杂度：O(1)O(1)O(1)
  • 时间复杂度：O(nlog2n)O(nlog_2n)O(nlog2​n)
  • 稳定性：不稳定

归并排序

1. 二路归并排序

将两个或两个以上的有序表组合成一个新的有序表。

  • 空间复杂度：O(n)O(n)O(n)
  • 时间复杂度：O(log2n)O(log_2n)O(log2​n) 一趟归并 O(n)O(n)O(n)，共 log2nlog_2nlog2​n 趟
  • 稳定性：稳定
private void mergeSort(int[] arr, int low, int high) {
    if (low < high) {
        int mid = (low + high) >> 1;
        mergeSort(arr, low, mid);
        mergeSort(arr, mid + 1, high);
        merge(arr, low, mid, high);
    }
}
// 写法一
private void merge(int[] arr, int low, int mid, int high) {
    int[] tmp = new int[arr.length];
    for (int i = low; i <= high; i++) {
        tmp[i] = arr[i];
    }
    int i, j, k;
    // 合并 [low, mid]...[mid + 1, high]
    for (i = low, j = mid + 1, k = i; i <= mid && j <= high; k++) {
        if (tmp[i] < tmp[j]) {
            arr[k] = tmp[i];
            i++;
        } else {
            arr[k] = tmp[j];
            j++;
        }
    }
    // 合并剩余元素
    while (i <= mid) {
        arr[k++] = tmp[i++];
    }
    while (j <= high) {
        arr[k++] = tmp[j++];
    }
}
// 写法二
private void merge(int[] nums, int low, int mid, int high) {
    int len = high - low + 1;
    int[] tmp = new int[len];
    // [low...mid] [mid + 1, high]
    int i = low, j = mid + 1, k = 0;
    while (i <= mid && j <= high) {
        tmp[k++] = nums[i] > nums[j] ? nums[j++] : nums[i++];
    }
    while (i <= mid) {
        tmp[k++] = nums[i++];
    }
    while (j <= high) {
        tmp[k++] = nums[j++];
    }
    System.arraycopy(tmp, 0, nums, low, len);
}

非比较排序

1. 计数排序

将输入的数据值转化为键存储在额外开辟的数组空间中。 作为一种线性时间复杂度的排序，计数排序要求输入的数据必须是有确定范围的整数。

局限性：

  1. 当数列最大最小值差距过大时，并不适用计数排序。
  2. 当数列元素不是整数，并不适用计数排序。
private static int[] countSort(int[] arr) {
    int max = Integer.MIN_VALUE;
    int min = Integer.MAX_VALUE;
    for (int num : arr) {
        max = Math.max(max, num);
        min = Math.min(min, num);
    }
    // 桶数组的大小为 最大值和最小值的差值
    int[] bucket = new int[max - min + 1];
    for (int value : arr) {
        // value 偏移位置 = value - min
        bucket[value - min]++;
    }
    // 对每个值加上之前的和 作为当前值
    // bucket[i] 表示 第 i 个元素的顺序
    for (int i = 1; i < bucket.length; i++) {
        bucket[i] += bucket[i - 1];
    }
    int[] res = new int[arr.length];
    // 获取原数组的每个元素的新位置
    for (int value : arr) {
        // 第 value - min 个元素的序号
        int pos = bucket[value - min] - 1;
        res[pos] = value;
        // 更新 第 value - min 个元素的新位置：用于重复元素的情况
        bucket[value - min]--;
    }
    return res;
}
  • 空间复杂度：O(n+k)O(n+k)O(n+k)
  • 时间复杂度：O(n+k)O(n+k)O(n+k)
  • 稳定性：稳定
  • 其排序速度快于任何比较排序算法。当 k 不是很大并且序列比较集中时，计数排序是一个很有效的排序算法。

2. 桶排序

将数据分到有限数量的桶里，每个桶再分别排序，最后输出每个桶的数据。

private static void bucketSort(int[] arr) {
    int bucketCount = arr.length;
    int min = arr[0];
    int max = arr[0];
    for (int value : arr) {
        min = Math.min(min, value);
        max = Math.max(max, value);
    }
    // 每个桶的大小
    double space = (double) (max - min) / (bucketCount - 1);
    // 初始化桶
    List<List<Integer>> buckets = new ArrayList<>();
    for (int i = 0; i < bucketCount; i++) {
        buckets.add(new ArrayList<>());
    }
    // 每个数据入桶
    for (int value : arr) {
        // 每个元素对应的桶序号
        int bucket = (int) ((value - min) / space) - 1;
        if (bucket < 0) {
            bucket = 0;
        }
        buckets.get(bucket).add(value);
    }
    // 对每个桶排序
    for (List<Integer> bucket : buckets) {
        Collections.sort(bucket);
    }
    // 输出桶内的每个元素
    int idx = 0;
    for (List<Integer> bucket : buckets) {
        for (int num : bucket) {
            arr[idx++] = num;
        }
    }
}
  • 空间复杂度：O(n+k)O(n + k)O(n+k)
  • 时间复杂度：
    • 最好：O(n)O(n)O(n)
    • 最坏：O(n2)O(n^2)O(n2)
    • 平均：O(n+k)O(n + k)O(n+k)
  • 稳定性：稳定

3. 基数排序

按照低位先排序，然后收集；再按照高位排序，然后再收集；依次类推，直到最高位。

private static void radixSort(int[] arr) {
    int max = arr[0];
    for (int num : arr) {
        max = Math.max(max, num);
    }
    List<List<Integer>> buckets = new ArrayList<>();
    // 余数 0 ~ 9 对应的桶
    for (int i = 0; i <= 9; i++) {
            buckets.add(new ArrayList<>());
    }
    // 当前排序位置
    int location = 1;
    while (true) {
        int radix = (int) Math.pow(10, (location - 1));
        // max < 需要取余的数，说明每一位都已排序完成
        if (max < radix) {
            break;
        }
        // 把每个元素放入对应的桶
        for (int num : arr) {
            // 计算余数
            int bucket = (num / radix) % 10;
            buckets.get(bucket).add(num);
        }
        // 取出桶内元素写回原数组
        int idx = 0;
        for (int i = 0; i <= 9; i++) {
            for (int num : buckets.get(i)) {
                arr[idx++] = num;
            }
            // 清空本次排序的桶
            buckets.get(i).clear();
        }
        location++;
    }
}
  • 空间复杂度：O(r)O(r)O(r)
  • 时间复杂度：O(d(n+r))O(d(n + r))O(d(n+r))
  • 稳定性：稳定

总结

每一次排序之后都能确定至少一个元素位置的排序方法包括：

  1. 选择排序：每次将最大的数放到最后。所以最大的数排一次序后位置就确定了。

  2. 冒泡排序：同选择排序。每一次排序最大的值位置确定。

  3. 快排：每一次排序 pivot 的位置确定。

  4. 堆排序：每一次排序时，都是将堆顶的元素和最后一个节点互换，然后调整堆，再将堆大小减1。所以每一次排序堆顶元素确定。
