vim 常用指令记录

命令模式

光标移动

cmd description
h 左移一个字符
l 右移
k 上移
j 下移
{n}{op} 10j 表示下移 10 行
{n}<Space> 向后移动 n 个字符
{n}<Enter> 向下移动 n 行
$ 移动到行首
0 移动到行尾
ctrl + f Page Down
ctrl + b Page Up
ctrl + d 下移半页
ctrl + u 上移半页
H 移动到这个屏幕的第一行的第一个字符
M 移动到这个屏幕的中央那一行的第一个字符
L 移动到这个屏幕的最后一行的第一个字符
gg 移动到文件第一行
G 移动到文件最后一行
{n}G 移动到第 n 行

剪切/删除

cmd description
x, X delete, backspace
{n}x 向后删除 n 个字符
dd 删除一行
{n}dd 删除 n 行
d1G 删除第一行到光标所在行
dG 删除光标所在行到最后一行
d0 删除光标到该行首部
d$ 删除光标所在处，到该行尾部(包括光标选中的字符)

复制

cmd description
y v 选中
yy 复制一行
{n}yy 复制 n 行
y1G 复制第一行到光标所在行
yG 复制光标所在行到最后一行
y0 复制光标到该行首部
y$ 复制光标所在处，到该行尾部(包括光标选中的字符)

粘贴

cmd description
p, P 粘贴在上/下一行
J 合并光标所在行和下一行

撤销、重做、重复

cmd description
u 撤销
ctrl + r 重做
. 重复上一个动作

末行模式

只记录不熟悉的

cmd description
ZZ 相当于 wq
ZQ 相当于 q!
:w {filename} save as
:r {filename} 将 filename 文件追加到光标后
